# nostalgiaTok
app that allows you to watch old TikToks like you're back in that era
link to working site: https://ddyson08.github.io/nostalgiatok/
hope you guys like it I spent really long on this :)

Dev started Nov 18, 2025

Most recent work done: May 12, 2026

You can use this code to spin up your own version of nostalgiaTok. Just download the file, decompress it, and then click on Index.html to open it. You can even use your own Google Account for the API by following the directions in GoogleScript.js.
